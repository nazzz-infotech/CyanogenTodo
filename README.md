### Start It as development server
```shell
# Mongodb compass required
# Npm required
# python required
# ------------------------------------------
# First Start Back-end
cd ./back-end
./venv./scripts/activate
py ./main.py
# Then start front end from other terminal window
cd ./front-end
npm run dev
```
